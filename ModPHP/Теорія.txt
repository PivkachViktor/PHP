Цілі числа (Integer): Це тип даних для представлення цілих чисел, наприклад, $age = 25;.

Дійсні числа (Float або Double): Вони використовуються для представлення чисел з плаваючою комою, наприклад, $price = 12.99;.

Рядки (String): Використовуються для представлення текстової інформації, наприклад, $name = "John Smith";.

Булевий тип (Boolean): Цей тип даних може приймати два значення: true (істина) або false (хибність). Наприклад, $isStudent = true;.

Масиви (Array): Масиви використовуються для зберігання декількох значень у впорядкованому списку. Наприклад, $colors = array("Red", "Green", "Blue");.

Об'єкти (Object): Об'єкти використовуються для створення складних структур даних за допомогою класів і об'єктів.

Ресурси (Resource): Цей тип даних використовується для представлення зовнішніх ресурсів, таких як відкриті файли або з'єднання з базами даних.

NULL: Значення null використовується для позначення відсутності значення.
